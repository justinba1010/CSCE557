# Elliptic Curve Point Arithmetic

## Usage

`python3 driver.py inputcurve`
Where `inputcurve` is a file with the following format for curve:

$$y^2 = x^3 + ax + b \mod m$$

`a b m1`

So for the curve: $y^2 = x^3 + x + 1 mod 31$, the file will have the contents of:
```
1 1 31
```
